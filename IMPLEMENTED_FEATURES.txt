# 🚀 Asteroid Logistics: Текущее состояние проекта

## 🛠️ Технологический стек
- **Frontend:** React + TypeScript + Vite
- **Состояние:** Zustand + Persist (LocalStorage)
- **Анимации:** Framer Motion + CSS Keyframes
- **Стилизация:** Tailwind CSS 4.0
- **Шрифты:** Play (основной), Fira Code (моноширинный)

## 💎 Экономика и Ресурсы

| Ресурс | Базовая цена | Редкость (спавн) | Ручная добыча |
| :--- | :--- | :--- | :--- |
| **Металл** | 1 CR | 60% | 1 клик = +2 ед. |
| **Лед** | 5 CR | 25% | 2 клика = +4 ед. |
| **Кристаллы** | 25 CR | 10% | 3 клика = +6 ед. |
| **Иридий** | 100 CR | 5% | 3 клика = +6 ед. |

### Склад и Логистика
- **Базовая емкость:** 100 единиц.
- **Транспорт:** Вылетает при заполнении минимум на 20%. Склад очищается мгновенно при вылете, кредиты начисляются сразу.
- **Оффлайн-доход:** При возвращении в игру рассчитывается добыча (макс. за 8 часов). Если склад переполнен, ресурсы распределяются пропорционально.

## 📡 Мини-игра: Радарное сканирование
Полноэкранный режим "Сапера" для разведки плотных залежей.

### Механика добычи в радаре:
- **Объем (Yield):** Каждая клетка с ресурсом дает **5% от макс. емкости склада** (но не менее 5 единиц).
- **Переполнение (Overflow):** Если склад полон, излишки продаются принудительно за **75%** от рыночной цены.
- **Опасности (Hazards):** Попадание в зону радиации отнимает **3 импульса**.

### Энергия радара:
- **Максимум:** 3 единицы.
- **Восстановление:** 1 единица каждые 5 минут.

### Апгрейды радара:
| Апгрейд | Эффект | Базовая цена | Множитель |
| :--- | :--- | :--- | :--- |
| **Батарея** | +2 импульса за сканирование | 200 CR | 2.0x |
| **Глубокий сканер** | Открывает Ice (L1), Crystal (L2), Iridium (L3) | 500 CR | 3.0x |
| **Ширина луча** | Увеличивает сетку (5x5 -> 6x6 -> 8x8) | 1000 CR | 4.0x |
| **Сонар** | Авто-открытие ресурсов на старте | 300 CR | 2.5x |

## 🎮 Дополнительные фишки
- **Разгон (Overclock):** Ускоряет добычу в 1.5 раза на 10 секунд.
- **Архив:** В меню улучшений можно посмотреть детальную статистику по всем открытым типам астероидов.
- **Камера:** Возможность приближения (zoom) и перемещения (pan) по главной сцене.
- **Визуальные эффекты:** Параллакс звезд, реактивные струи двигателей, всплывающие цифры добычи, тряска экрана при продаже.
